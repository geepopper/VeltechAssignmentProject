VeltechAssignmentProject was created as a demo project to highlight how selenium framework libraries can be utilised for automated interaction with web applications.
NB: There is a bit of a rush to get this project out. As such, some necessary code refactoring will later be made.

To run this project on a clean install window, follow the following steps:


1. Download and install Visual Studio on your machine
2. Install Git on your machine
3. Clone this project
4. Click on the .sln file to open the project solution on Visual studio.
